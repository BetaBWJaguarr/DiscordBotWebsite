Commands

Here you can view and manage all available commands.

/ban

Bans a user from the server.

Usage: /ban <username> [duration] [reason] [delete_history_message_duration]

Username: The user to ban. Required
Duration: How long the ban will last. Optional
Reason: Reason for the ban. Optional
Delete History Message Duration: Duration for deleting history messages. Optional
/kick

Kicks a user from the server.

Usage: /kick <username> [reason]

Username: The user to kick. Required
Reason: Reason for kicking the user. Optional
/mute

Mutes a user in the server.

Usage: /mute <username> <duration> [reason]

Username: The user to mute. Required
Duration: How long the mute will last. Required
Reason: Reason for muting the user. Optional
/verify

Verifies a user with a specific level and status.

Usage: /verify <username> <level> <status>

Username: The user to verify. Required
Level: Verification level. Required
Status: Verification status. Required
/autorole

Sets an automatic role for new users.

Usage: /autorole <role>

Role: The role to assign automatically. Required
/punishment-search

Searches for punishments based on criteria.

Usage: /punishment-search <type> <predicate> <value>

Type: Type of punishment to search for. Required
Predicate: Condition to match the punishment. Required
Value: Value to match the predicate. Required
/antiswear list

Lists all antiswear words.

Usage: /antiswear list

/antiswear add

Adds a new antiswear word.

Usage: /antiswear add <word>

Word: The word to add to the antiswear list. Required
/antiswear remove

Removes an antiswear word.

Usage: /antiswear remove <word>

Word: The word to remove from the antiswear list. Required
/setlanguage

Sets the language for the server.

Usage: /setlanguage <language>

Language: The language to set. Required
